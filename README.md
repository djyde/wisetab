# Wisetab

Readwise daily review on your browser's new tab page.

![Frame 12](https://github.com/djyde/wisetab/assets/914329/77841fee-8160-4fc5-9a08-0f26e837142c)

## Getting Started

First, run the development server:

```bash
pnpm dev
# or
npm run dev
```

## Making production build

Run the following:

```bash
pnpm build
# or
npm run build
```
